# AWS Wazuh Cloud Security Lab

## Overview

This project demonstrates the implementation of a cloud-based security monitoring environment using AWS and Wazuh SIEM.

## Technologies Used

- AWS EC2
- AWS VPC
- Wazuh SIEM
- DVWA Application
- CloudWatch
- VPC Flow Logs
- Amazon S3
- Linux
- Windows

## Implemented Use Cases

- SSH Brute Force Detection
- Port Scan Detection
